# SIGOPEA - Gerenciamento de obras e projetos

[![License](https://img.shields.io/badge/TCC-FACULDADE%20SENAC-yellow.svg)](http://187.7.106.14)
[![License](https://img.shields.io/badge/Miguel%20Oliveira-Aluno-blue.svg)](http://187.7.106.14:7963/u/Miguel)

Essa aplicação permite que os profissionais gerenciem seus trabalhos, podendo criar agendamentos, adicionar observações, enviar e-mail e gerar gráficos e relatórios gerenciais, e, ainda, permite que os clientes acompanhem o andamento de sua obra.

A ideia é que, através de um aplicativo Android, possa haver uma melhor comunicação entre profissional e cliente, onde o cliente irá acompanhar o andamento de sua obra através de imagens e comentários fornecidos pelo profissional e, com uma aplicação Desktop, o profissional possa gerenciar seus trabalhos de forma mais eficaz, além de conseguir gerar gráficos e relatórios.
## Tecnologias utilizadas

* C#
* Android
* SQLite
* SQLServer
* MySQL
* PHP
* Entity Framework

## Configurações

1. Edite o arquivo `App.config` e, na connectionString, informe o caminho do arquivo(.mdf) do seu banco de dados.
2. Salve e rode a aplicação.
